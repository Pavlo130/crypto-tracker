# Crypto Tracker

Crypto Tracker — це вебзастосунок для відстеження криптовалют, створений за допомогою React.

## Можливості

- Пошук криптовалют
- Фільтр монет, що ростуть та падають
- Перегляд актуальної ціни
- Детальна сторінка кожної монети
- Графік зміни ціни (24H / 7D / 30D / 90D)

## Технології

- React
- Vite
- React Router
- Recharts
- CoinGecko API
- CSS
